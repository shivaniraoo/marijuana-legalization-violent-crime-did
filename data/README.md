## Data Overview

This folder contains cleaned, analysis-ready datasets used in the Difference-in-Differences analysis.

### Data Sources
- FBI Crime Data Explorer (CDE): Monthly violent crime counts by state
- U.S. Census Bureau: State population estimates

### Notes
- Raw FBI data was cleaned, aggregated, and normalized to crime rates per 100,000 residents.
- Only final datasets are included to improve reproducibility and clarity.
- Raw data can be re-downloaded from the FBI Crime Data Explorer.
